Instalación:
    -En el archivo python ".py" modificar:
        user: pones tu email de discord
        passwd: pon tu contraseña de discord
        canal: pon el url del canal de texto
        hour: horas entre mensajes (4 horas por defecto)
        driver_path = 'C:\\Users\\USUARIO\\Desktop\\chromedriver.exe' : modificar path donde este situada la carpeta que contenga chromedrive.exe separando la ruta por \\
    -Al iniciar el .py es recomendable hacerlo con la versión de Python 3.9 de la Windows Store que presenta menos fallos en la ejecución de esta herramienta

Uso:
    2 Modos de uso:
        Auto = Este modo realiza segun el intervalo de tiempo la tarea Work o el conjunto de todas (Work, Slut, Crime)
        Manual = En este modo se pueden realizar estas acciones manualmente sin acceder a discord y a alta velocidad sin que te roben
            Cada vez que se realice una opción te preguntara si quieres parar el programa seleccionar la opcion "1.NO" para continuar poniendo comandos rápidos

Errores:
    No se abre
        Prueba a instalar Python 3.9 desde la windows store y seleccionar "abrir con" y seleccionas esta aplicacion
    Eror al abrir
        Si no aparece el título es porque no esta localizado el driver de navegador, asegurate de haber cambiado bien la ruta en el apartado de driver_path
    Error al seleccionar las opciones
        El email y contraseña no son correctos y no se han podido realizar las acciones, compruebalos.
    Como se cierra
        El programa deberai cerrrarse al cerrar la consola, si no es así pulsando ctrl+c deberia parar
    El título se muestra mal
        La aplicación esta pensada apra ser ejecutada en consola maximizada
